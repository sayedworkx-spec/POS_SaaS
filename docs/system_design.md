# POS SaaS

## Roles

### Platform Admin

Can manage:
- Companies
- Subscriptions
- All Users

### Company Admin

Can manage:
- Products
- Sales
- Cashiers

### Cashier

Can:
- Create Sales
- View Products

Cannot:
- Manage Users
- Manage Subscription
